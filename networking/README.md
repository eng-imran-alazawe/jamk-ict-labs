# Networking Labs

This folder contains practical networking exercises completed during ICT Engineering studies.

## Topics
- Basic network concepts and architectures
- IP addressing and subnetting
- Routing fundamentals
- DHCP configuration and usage
- Network Address Translation (NAT)
- Basic firewall and connectivity testing

## Tools and Environments
- Virtual machines
- Linux and Windows networking tools
- Isolated lab environments for testing

## Goal
Develop a practical understanding of how computer networks are configured, managed, and troubleshooted in IT environments.
