# JAMK ICT Labs

This repository contains practical labs and hands-on exercises completed during ICT Engineering studies at JAMK University of Applied Sciences.

The focus is on building a solid foundation in IT operations through real technical practice rather than theory only.

## Topics Covered
- Linux and Windows administration
- Networking fundamentals (routing, DHCP, NAT)
- Scripting and automation with Bash and PowerShell
- Cybersecurity fundamentals and secure configurations
- Virtual machines and WSL-based lab environments

## Structure
Content is organized by topic. Each folder contains labs, notes, configurations, or documentation related to that area.

## Goal
Demonstrate hands-on technical skills relevant to junior IT and trainee roles.
